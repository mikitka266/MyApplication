# <b>MyApplication</b>
My personal first project for Food Company
Мой проект представляет собой ИТ-решение для индустрии общественного питания. Суть проекта будет заключаться в том, что это будет некий аналог существующих решений по типу iiko и специальных модулей для 1с, но на бесплатной основе для небольших предприятий и ИП.
Мой модуль, в идеале, приложение, позволит собственнику или руководителю бизнеса видеть расход продуктов по кухне, кассовую выручку и их сопоставлять в онлайн режиме.
Каким образом этот модуль будет реализован.
Здесь будет создан многопользовательский режим
Повар, отвечающий за продукты, и качество приготовленных блюд, в его смену, которому будет присвоен свой айдишник, соответствующий ФИО сотрудника
Бармен, отвечающий за напитки, алкогольную продукцию и бакалею, соответственно со своим АйДи.
Официант, который отвечает за уровень сервиса, показателями которого будут являться время отдачи блюд, и средний чек в заведении.

Все данные будут записываться в файл.
Блюда и напитки будут вынесены в отдельные интерфейсы, а конкретные блюда и напитки буту расписаны как методы отдельные.
В третьем файле будут остатки по складу, вычисленные с помощью вычитания текущих продаж.
Также будет файл Инвентаризации, который будет сверяться ежемесячно для кухни, и еженедельно для бара.

В соответствии, с итогами инвентаризации, будет вычислен kpi либо выявлены штрафы, для сотрудников, отвечающих за свои подразделения. 
При значительных излишках в продуктах, будет рекомендация пересмотреть, конкретные рецептуры, в которых присутствуют эти продукты, также при недостаче, будет рекомендовано провести контрольную проработку блюда, для выявления потерь.
